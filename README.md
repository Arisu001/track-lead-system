# Track Lead System

Config only in solo

### Install
Copy the `track-lead-system` into the `datapack` folder of your folder.


## Usage
### How to use
```
Use compass to open the menu
```

### How to modify properties
```
Right click on the text to modify
```

### Uninstall
```
Instant double right-click on the barrier item
```
